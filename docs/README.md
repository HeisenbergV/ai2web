## 🚀 快速开始

### 📖 首次使用必读

**👉 [使用指南](./USAGE_GUIDE.md)** - **强烈推荐先阅读此文档！**

详细说明如何使用这套文档系统，包括：
- 如何开启新会话
- 如何描述需求
- 最佳实践
- 示例对话

### 对于用户

1. **新会话开始时：**
   ```
   请先阅读 docs/AI_SYSTEM_PROMPT.md 和 docs/QUICK_START.md
   ```

2. **开发新功能时：**
   - 参考 `docs/DEVELOPMENT_GUIDE.md` 确保代码规范
   - 使用 `docs/AI_CONVERSATION_TEMPLATE.md` 中的模板描述需求

### 对于 AI

1. **新会话开始时：**
   - 首先阅读 `docs/AI_SYSTEM_PROMPT.md` 理解项目架构
   - 阅读 `docs/QUICK_START.md` 快速了解关键文件

2. **开发过程中：**
   - 参考 `docs/DEVELOPMENT_GUIDE.md` 确保代码符合规范
   - 遵循编码规范和最佳实践

## 📋 文档使用指南

### 场景 1：添加新功能

1. 阅读 `AI_SYSTEM_PROMPT.md` 了解架构
2. 参考 `QUICK_START.md` 查看常见任务
3. 使用 `AI_CONVERSATION_TEMPLATE.md` 中的模板描述需求
4. 开发时参考 `DEVELOPMENT_GUIDE.md` 确保规范

### 场景 2：修复 Bug

1. 使用 `AI_CONVERSATION_TEMPLATE.md` 中的调试模板
2. 参考 `DEVELOPMENT_GUIDE.md` 检查代码规范

### 场景 3：优化代码

1. 参考 `DEVELOPMENT_GUIDE.md` 中的规范
2. 使用 `AI_CONVERSATION_TEMPLATE.md` 中的优化模板
